## ğŸ¨ Art Style Guidelines

### Color Palette
- **Background**: Dark grays (#0f0f12, #17171b)
- **Accent Orange**: #ffb74d (primary UI color)
- **Accent Blue**: #5ec8f2 (secondary/prestige)
- **Good Green**: #8bc34a (success)
- **Danger Red**: #ff6b6b (warnings/reset)
- **Text**: #eeeeef (light), #b3b3bd (muted)

### Style Notes
- **Pixel size**: Clean, crisp pixels (no anti-aliasing on edges)
- **Outlines**: 1-2px dark outlines for contrast
- **Shading**: Simple 2-3 tone shading
- **Dogs**: Cute, friendly, expressive
- **Buildings**: Industrial but warm/inviting
- **Banners**: Horizontal scenes showing progression

### Animation Notes
- **Frame rate**: 2-6 FPS for idle animations
- **Loop**: All animations should loop smoothly
- **Frames**: Keep to 2-4 frames per animation for file size

## ğŸ’¡ Tips for Artists

1. **Start with simple shapes** - Dogs can be very simple pixel art
2. **Banners tell a story** - Show progression visually
3. **Consistent style** - Keep pixel density similar across all assets
4. **Test at scale** - View at actual game size (many will be small!)
5. **Transparent backgrounds** - Save as PNG with transparency
6. **Name files clearly** - Follow the naming convention exactly
